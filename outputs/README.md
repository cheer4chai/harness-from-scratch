# Outputs

Reusable artifacts produced by lessons.

## Directories

- `prompts/` - prompts that can be pasted into an assistant.
- `skills/` - `SKILL.md`-style task instructions.
- `agents/` - agent definitions with role, instructions, and allowed tools.
- `mcp-servers/` - small local servers or server definitions used by MCP lessons.

Keep outputs small and tied to a lesson. If an artifact needs supporting code, place that code in the lesson folder and link to it from here.
