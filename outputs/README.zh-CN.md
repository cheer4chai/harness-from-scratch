# Outputs

这里存放课程章节产出的可复用 artifacts。

## 目录

- `prompts/` - 可以粘贴到 AI assistant 中使用的 prompt。
- `skills/` - `SKILL.md` 风格的任务指令。
- `agents/` - 包含角色、指令和 allowed tools 的 agent 定义。
- `mcp-servers/` - MCP 章节使用的小型本地 server 或 server 定义。

保持 artifact 小而聚焦，并明确绑定到某个课程章节。如果 artifact 需要配套代码，把代码放在对应 lesson 目录中，并从这里链接过去。
